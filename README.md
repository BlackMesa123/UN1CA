<p align="center">UN1CA <i>(/Ëˆu.ni.ka/)</i> is a work-in-progress custom firmware for Samsung Galaxy devices.</p>

# How to build
```bash
git clone --recurse-submodules https://github.com/BlackMesa123/UN1CA.git && cd UN1CA
. ./buildenv.sh <device>
run_cmd make_rom
```
